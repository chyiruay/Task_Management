# Multi-User Task Management System

这是一个旨在提高团队协作效率的集中式任务管理平台。它支持多用户环境下的任务创建、分配、跟踪及状态更新。

## 🎯 项目目标 (Objective)
建立一个集中的应用程序，使多个用户能够：
* 高效地管理工作与任务。
* 实现任务的全生命周期管理（创建、跟踪、完成）。
* 透明化团队进度，通过状态更新和截止日期确保准时交付。

## ✨ 功能需求 (Functional Requirements)

### 1. 仪表盘视图 (Dashboard)
* 提供可视化摘要，展示不同用户的任务分布和完成情况。

### 2. 任务管理 (Task Management)
* **基础操作:** 创建、编辑、删除和查看任务。
* **详细信息:** 包含任务描述、截止日期 (Due Dates)。
* **团队协作:** 支持将任务指派给其他团队成员。

### 3. 状态与优先级 (Status & Priority)
* **状态追踪:** New, In Progress, UAT, Completed, Closed。
* **优先级管理:** High, Medium, Low。

### 4. 用户权限 (Authentication & Roles)
* 集成用户登录功能。
* 支持基于角色的权限控制（Roles），确保数据安全与操作合规。
